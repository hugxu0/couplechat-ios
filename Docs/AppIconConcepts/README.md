# App Icon Concepts

## Cat Heart Bubble — current

The user-selected pink cat-ear speech bubble with a heart is now the active AppIcon. Its supplied rounded-corner backdrop is normalized to a full square before iOS applies the system icon mask.

## Sakura Bubbles

Two overlapping speech bubbles form a quiet conversation, with a small heart at the overlap. The sakura gradient follows the app's new default theme and remains legible at small sizes.

## Midnight Orbit

A night-blue mark with a soft orbit and two message bubbles. It connects the existing “星夜” wallpaper mood with the long-night-chat identity.

## Cat Whisper

An orange cat inside a speech bubble. It gives the 大橘 character a stronger brand role while keeping the mark simple enough for an app icon.
